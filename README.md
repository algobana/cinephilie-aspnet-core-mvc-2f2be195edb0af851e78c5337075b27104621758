# cinephilie-aspnet-core-mvc

Cinephilie is a university project developed as part of a web programming course. The platform allows users to list, rate, and review movies in a personalized and interactive way. With Cinephilie, users can keep track of the films they've watched or plan to watch, write detailed reviews, and explore new titles based on community feedback.
